Sobre

Este site foi desenvolvido para disponibilizar dados de uma estação meteorológica local em tempo real.

Aqui você pode acompanhar informações como temperatura, umidade do ar e outras variáveis ambientais, coletadas automaticamente por sensores conectados a um sistema eletrônico.

O objetivo é fornecer dados úteis para estudos, curiosidade e acompanhamento das condições climáticas da região.

Este projeto também possui caráter educacional, demonstrando o uso de tecnologias como microcontroladores, sensores e integração com a internet.
